# ⚖️ Islamic Inheritance Calculator (نظام المواريث)

"An interactive **Python** application designed to calculate legal inheritance shares according to Islamic Sharia law."

## 🌟 Project Overview
This tool simplifies the complex process of 'Mawarith' (Inheritance) by guiding the user through a series of questions about the deceased's family members. It applies the legal rules of exclusion (Hajb) and fractional distribution (1/2, 1/4, 1/8, 1/6, 1/3) automatically.

## ✨ Features
* **User-Friendly Interface:** Command-line interactive prompts to identify heirs.
* **Comprehensive Logic:** Handles various scenarios for spouses, parents, grandparents, and siblings.
* **Sharia Evidence:** Provides Quranic verses from Surah An-Nisa as a reference for the calculated shares.
* **Accuracy:** Built using conditional logic to ensure each heir receives their exact legal portion.

## 🛠 Tech Stack
* **Language:** Python 3.x
* **Architecture:** Procedural logic based on Islamic jurisprudence (Fiqh).
